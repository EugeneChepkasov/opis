ОПИСЬ ИМУЩЕСТВА — как установить
================================

Состав:
  index.html            — само приложение (работает и как одиночный файл)
  sw.js                 — офлайн-кэш (service worker)
  manifest.webmanifest  — манифест для установки на главный экран
  icon-192.png, icon-512.png — иконки

ВАРИАНТ 1 (рекомендуется): выложить на свой сервер
--------------------------------------------------
1. Загрузите все файлы папки в каталог на VPS, например:
     /var/www/opis/
2. В nginx добавьте location (или отдельный поддомен):
     location /opis/ {
       alias /var/www/opis/;
       index index.html;
     }
   Обязательно HTTPS — офлайн-режим (service worker) работает
   только по https.
3. Откройте адрес на телефоне в Chrome, например:
     https://ваш-домен/opis/
4. Меню Chrome (⋮) → «Добавить на главный экран» → «Установить».
   Появится иконка, приложение откроется без адресной строки.
5. После первого открытия всё кэшируется — дальше работает
   полностью без интернета.

Коллегам достаточно прислать ссылку — каждый ставит себе так же.

ВАРИАНТ 2: без сервера, просто файлом
-------------------------------------
Можно переслать index.html (одного файла достаточно) в Telegram,
скачать и открыть в Chrome. Считать и копировать списки будет
работать, но есть два минуса:
  - нельзя «установить» с иконкой и офлайн-кэшем;
  - сохранённые списки могут не пережить повторное открытие файла
    из загрузок (Android открывает такие файлы по временному пути).
Поэтому для постоянной работы лучше вариант 1.

ДАННЫЕ
------
Списки хранятся локально в браузере каждого телефона
(localStorage) — у каждого сотрудника свои, между устройствами
не синхронизируются. Кнопка «Скопировать текстом» в сохранённом
списке открывает системное меню «Поделиться» — оттуда список
можно отправить в Telegram/почту.
